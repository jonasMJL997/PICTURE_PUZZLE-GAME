## 🧩 What is this?

A browser-based **picture puzzle game** where players upload their own personal photos and solve them as sliding tile puzzles. Fully configurable — choose your grid size, difficulty, and image. No downloads, no installs — runs entirely in the browser.

---

## ✨ Features

- 📸 **Custom image upload** — use any photo from your device
- ⚙️ **Configurable difficulty** — adjust grid size (3×3, 4×4, 5×5)
- 🔀 **Auto-shuffle** — randomised tile placement on each game start
- ✅ **Win detection** — automatically recognises when the puzzle is solved
- ⏱️ **Move counter** — tracks how many moves it took to solve
- 📱 **Responsive design** — works on desktop and mobile browsers

---

## 🎮 How to Play

1. Open the [live demo](https://jonasmjl997.github.io/PICTURE_PUZZLE-GAME) or clone and run locally
2. Click **"Upload Image"** and select any photo from your device
3. Choose your preferred grid size
4. Click **"Start Game"** — tiles will be shuffled
5. Click adjacent tiles to slide them into position
6. Recreate the original image to win!

## 💡 Key JavaScript Concepts Used

- **FileReader API** — reads the uploaded image file into the browser
- **Canvas API** — splits the image into puzzle tile segments
- **DOM Events** — handles click-to-slide game interactions
- **Array shuffling** — Fisher-Yates algorithm for fair tile randomisation
- **State management** — tracks tile positions and checks win condition
